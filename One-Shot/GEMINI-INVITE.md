## Lesson Learned: Model Switching

### The Mess We Recovered From

| What Happened | Impact |
|---------------|--------|
| Session started on GPT-4.1 | Wrong model, different outputs |
| Default budget caused auto-switch | Human didn't notice initially |
| Had to redo under Claude | Time lost, lesson learned |

### Prevention

| Step | Action |
|------|--------|
| **Before session** | Check budget settings in GitHub Copilot |
| **Confirm model** | Ask "What model are you?" at start |
| **Document model** | Every file signed with model name |
| **If switch detected** | Stop, document, decide: redo or continue |

### For Gemini Session

```
⚠️ BEFORE STARTING GEMINI SESSION:
1. Open GitHub Copilot settings
2. Verify budget allows Gemini
3. Start new session
4. Confirm: "What model are you?"
5. If not Gemini: STOP, fix settings, restart
```
